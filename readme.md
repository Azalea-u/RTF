# Real-Time-Forum App:

## Description:

The Real-Time-Forum App is a web application that allows users to create and view posts in a forum. Also include a chat feature for real-time communication between users.

## Features:

- Create posts
- Comment on posts
- Like posts
- View posts
- Real-time chat feature
